# 🛡️ Phishing Guard Pro

A web-based cybersecurity tool that detects phishing websites using multiple security layers.

---

## 🎯 Features
- URL safety scanning
- VirusTotal analysis
- SSL certificate check
- Domain age detection
- Risk scoring system

---

## ⚙️ How it Works
1. User enters a URL
2. System runs multiple checks:
   - VirusTotal API
   - SSL validation
   - Domain age analysis
   - Keyword detection
3. Final risk score is generated

---

## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- Security APIs
